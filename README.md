# Readable

This is a Udacity Readable Project from Udacity React Nanodegree.

## To run this application

    - `git clone https://github.com/nybragaramos/readable.git`
    - `cd readable`
    - `cd frontend`
    - `yarn install`
    - `yarn start`

Open http://localhost:3000 to view it in the browser.
